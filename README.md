IP Address Tracker
Web application to view details about an IP address.

Features:
1. See their own IP Address on the map on the initial page load
2. Search for any IP addresses or domains and see the key information and location
3. View the optimal layout for each page depending on their device's screen size
4. See hover states for all interactive elements on the page

Reflection:
Development process: The first step for me was to create a replica of the design that was provided without any functionality. This involved using HTML and CSS, both of which we were using sometime ago and therefore I need some brushing up on. I opted to use internal CSS and include the JS code in the HTML file to minimize the back and forth initially but later separated them into individual files to adhere to best practices.

Challenges faced: Re-familiarizing myself with CSS after about a 2-week break. While it was easy to pick it back up, I have always been a little shaky with my grasp of CSS - I occassionally get the feeling that the house of cards I have built will crumble if I shift something by a few pixels. More practice would certain alleviate this feeling.

Solutions implemented: 1) Bootstrap, 2) Leaflet, 3) IPify, 4) Accessibility features, 5) Error Handling, 6) Fetch(), 7) Event listener

Potential improvements: Refactoring the code to remove unnecessary lines of CSS code, possible bugs related to UI elements overlapping on a devices with smaller resolutions, etc.
